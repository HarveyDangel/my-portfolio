## Deployment

This portfolio is deployed on **Vercel** with automatic deployments on every push to `main`.

### Deploy your own

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

**Steps:**

1. Fork or clone this repository
2. Push to your GitHub account
3. Import the repo in [Vercel](https://vercel.com/new)
4. Set any required environment variables in the Vercel dashboard
5. Deploy вЂ” Vercel handles the rest

## Getting Started

```bash
# Clone the repository
git clone https://github.com/HarveyDangel/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
